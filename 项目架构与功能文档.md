# Simple Server 架构与功能文档

## 一、项目概述

**Simple Server** 是一个轻量级 PHP RESTful API 服务框架。采用中间件链 + 服务-仓库分层架构，内置频率限制、RBAC 权限管理、推荐系统、考勤/加班/薪资计算、以及自动化爬虫等完整功能模块。配套 uni-app（Vue 3）前端和管理后台。

## 三、架构设计

### 分层架构

```
请求（HTTP Request）
       │
       ▼
┌──────────────────┐
│   CORS 中间件     │  跨域白名单 + OPTIONS 预检
└────────┬─────────┘
         ▼
┌──────────────────┐
│   日志中间件      │  请求/响应 JSON 日志 + 毫秒计时
└────────┬─────────┘
         ▼
┌──────────────────┐
│   错误中间件      │  try/catch 全局异常捕获
└────────┬─────────┘
         ▼
┌──────────────────┐
│   路由层          │  URI 解析 → 分发到处理器
└────────┬─────────┘
         ▼
┌──────────────────┐
│   服务层          │  业务逻辑、权限验证、流程编排
└────────┬─────────┘
         ▼
┌──────────────────┐
│   仓库层          │  SQL 构建、PDO 执行
└────────┬─────────┘
         ▼
     MySQL 数据库
```

### 统一路由入口 (v2.2.0)

所有 `/api/*` 请求通过 `.htaccess` rewrite 进入 `index.php`：
```
/api/login.php         ──→  index.php?__route=login.php
/api/auth/login        ──→  index.php?__route=auth/login
/api/content.php?id=1  ──→  index.php?__route=content.php&id=1
```

路由分发策略：
| 路由类型 | 示例 | 处理方式 |
|----------|------|----------|
| RESTful 核心路由 | `auth/login`, `users/1`, `content` | 直接调用 Service 层（无重复 DB 连接） |
| 传统文件路由 | `follow.php`, `messages.php` 等 89 个 | `require` 原文件（中间件已就绪，header 可安全重复） |
| RESTful + query fallback | `/users.php?id=1` | `$_GET['id']` 自动兼容

## 六、响应格式

所有接口统一：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": { ... }
}
```

错误响应：

```json
{
  "code": 400,
  "message": "错误描述"
}
```

HTTP 状态码：
- `200` 成功
- `201` 创建成功
- `204` OPTIONS 预检
- `400` 参数错误
- `401` 未认证
- `403` 权限不足
- `404` 资源不存在
- `405` 方法不允许
- `429` 频率限制
- `500` 服务器错误

## 七、安全机制

| 措施 | 实现位置 |
|------|----------|
| bcrypt 密码加密 | login.php, register.php, UserService |
| 全量 PDO 预处理 | 所有 SQL 查询均使用 prepare + bind |
| 频率限制 | RateLimiter — login(5/5min), register(3/10min) |
| 管理员 Session 验证 | 所有 admin_*.php 统一检查 `$_SESSION['is_admin']` |
| 文件上传白名单 | 类型 + 大小限制 |
| 路径穿越防护 | basename() 防护 (download_proxy.php) |
| CORS 白名单 | cors_headers.php + Config.php .env 配置 |
| Token 认证 | login 成功返回 64 字符随机 hex token |
| 错误信息脱敏 | 生产环境不应暴露堆栈 |

## 八、特色功能详解

### 1. 用户画像与推荐（user_profile.php）

- **用户画像**：基本信息 + 加权行为统计 + 偏好标签 + 偏好内容
- **行为权重**：content_click=5, recommend_click=4, page_stay=1
- **内容相似度**：Jaccard 标签相似度 + 标题/正文词汇 + 分类匹配（4 维加权）
- **个性化推荐**：偏好标签 → LIKE 匹配 → 互动+新鲜度排序

### 2. 推荐算法管理系统（admin_recommendations.php）

- 4 种算法：协同过滤(0.3) / 基于内容(0.25) / 流行度(0.25) / 混合(0.2)
- A/B 测试框架：创建测试 → 分流量 → 对比 CTR/转化率
- 实时监控：最近 10 分钟展示/点击趋势
- 手动推荐：指定用户/内容/优先级/过期时间

### 3. RBAC 角色权限（admin_permissions.php）

- 4 个默认角色：super_admin → admin → editor → user
- 12 个权限项：manage_users, manage_content, manage_messages 等
- 可动态创建/编辑/删除角色并分配权限
- 用户-角色关联管理

### 4. 考勤与薪资（attendance.php + overtime.php）

- 考勤打卡：上班 `check_in` / 下班 `check_out`，实时计算工时
- 加班记录：时薪 × 小时 × 倍数
- 薪资总览（江苏标准）：底薪 + 奖金 + 绩效 + 加班费 - 五险一金 - 个税
- 五险一金可配置比例（养老 8% / 医疗 2% / 失业 0.5% / 公积金 8%）
- 个税采用累计预扣法（7 级超额累进税率）

### 5. 自动化爬虫

- **抖音热榜**：curl 抓取 → 格式化 Markdown → 自动发布到 articles 表
- **图片抓取**：picsum 随机图 → 自动发布
- 均含降级策略（API 失败时使用模拟数据）

### 6. AI 代理（ai_proxy.php）

- 转发请求到本地 LLM 服务 `192.168.1.10:8080`
- 支持 OpenAI 兼容 API 格式

- PHP 扩展：PDO_MySQL, fileinfo, gd, curl, openssl

### 安装步骤

```bash
# 1. 克隆
git clone git@github.com:jackchenjiufu/Simple.git
cd Simple

# 2. 安装依赖
composer install

# 3. 配置
cp server/config/.env.example server/config/.env
# 编辑 .env 填入数据库信息

# 4. 导入数据库
mysql -u root -p < server/sql/doo-app.sql

# 5. 目录权限
chmod 755 server/uploads/
chmod 755 server/downloads/
chmod 755 server/logs/
chmod 600 server/config/.env

# 6. 初始化
curl http://localhost/server/api/init_database.php
curl http://localhost/server/api/reset_admin.php  # 管理员密码 → admin123
```

### Apache

```apache
<VirtualHost *:80>
    DocumentRoot /var/www/simple
    ServerName your-domain.com
    <Directory /var/www/simple>
        Options -Indexes +FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```

### Nginx

```nginx
server {
    listen 80;
    root /var/www/simple;
    location ~ \.php$ {

        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }
    location /server/downloads/ {
        # APK 下载走 download_proxy.php 代理
    }
}
```
